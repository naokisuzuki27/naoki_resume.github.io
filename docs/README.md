#### Wordpress(EC2)投稿データのRDSへの移行 （2024年7月～2024年8月）

**担当**
- 構成把握から移行方法の策定、移行作業まで担当

**業務詳細**  
自社で運用しているWordPressの投稿データを、EC2インスタンスからRDS (MySQL) へ移行するプロジェクトにおいて、構成把握から移行方法の策定、移行作業まで一貫して担当しました。

移行の背景には、パフォーマンスの向上や運用の効率化を目的としており、EC2上のMariaDBをRDS(MySQL)に移行することで、データベースのスケーラビリティや可用性を改善することが期待されました。

移行後は、RDSにより運用の負荷が軽減され、パフォーマンスも向上しました。さらに、データベース管理の手間が削減され、今後の運用において柔軟にスケールできる体制を構築しました。

**環境**
- **インフラ基盤**: AWS
- **OS**: AmazonLinux2023
- **DB**: AWS RDS (MySQL)
- **サービス**: VPC, EC2, ALB, Route53

**役割/規模**: メンバー / 1名  
**担当**: 技術選定、構築
